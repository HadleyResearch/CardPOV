CardPOV Project Roadmap:
CardPOV PCB:
1) CardPOV v2.0		(USB Programmability, SMD/PTH dual-mode LED pads, choice of LED colors)
2) CardPOV v3.0		(USB battery recharging, increased SMD LEDs, power for increased LED count)
3) CardPOV v4.0		(External control access, POV intensity control)
4) CardPOV v5.0		([Experimental], Integrated touch-sensing capabilities to replace pushbuttons)
5) CardPOV v6.0		([Experimental], Bluetooth reprogrammability)
6) CardPOV v7.0		([Experimental], Accelerometer/Direction sensing support)
7) CardPOV v8.0		([Experimental], SDCard Support)
8) CardPOV v9.0		([Experimental], Full RGB Color Control)

CardPOV Firmware:
1) Firmware v2.0	(USB Bootloader, C-code port, improved multiple string support)
2) Firmware v3.0	(Battery Management Code, Unicode Support)
3) Firmware v4.0	(External POV triggering code, digital POV intensity control)
4) Firmware v5.0	(Touch-sensing code)
5) Firmware v6.0	(Bluetooth support)
6) Firmware v7.0	(Accelerometer code, POV control priority access code)
7) Firmware v8.0	(SDCard code)
8) Firmware v9.0	(RGB control code)

CardPOV Bracket Mount:
1) Bracket v1.0		(Basic Operational Model)

CardPOV PC Host:
1) CardPOV Host v1.0	(USB Driver, USB Programming Capability, firmware reprogramming capability)
2) CardPOV Host v1.5	(GUI)
3) CardPOV Host v2.0	(Unicode support)
4) CardPOV Host v3.0	iPhone/Droid port, Bluetooth driver
5) CardPOV Host v4.0	RGB Color control support
