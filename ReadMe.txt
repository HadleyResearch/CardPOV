Git repository for the CardPOV project. The CardPOV project is an open-source
persistence of vision device designed to be pocketable and roughly the size of
a business card. Included in this repository are relevant documentation, source
code, and pcb layout files necessary to recreate the CardPOV project. These files
are released to the general public free of charge under a Creative Commons 
BY-SA-3.0 license (http://creativecommons.org/licenses/by-sa/3.0/). If you do use
these files in one of your projects, please let us know, as we would love to 
feature your project on our website.
